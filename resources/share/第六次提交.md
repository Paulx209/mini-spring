=====================资源=====================

1.Resource接口，提供getInputStream()接口方法，供实现类实现。

2.三个资源实现类，分别为 ClassPathResource 类(类路径下的资源)、UrlResource类（URL对应的资源）、FileSystemResource类（文件系统中的资源）。

3.接口方法的最终目的就是返回一个InputStream字节流。

对应的属性分别为String path 、 URL url 、 File file

=====================资源加载器=====================

1.ResourceLoader接口，提供getResource()方法，其实就是获取Resource类

2.一个资源加载器实现类，DefaultResourceLoader类，getResource()方法中，根据参数location分三种情况判断，classpath路径下的资源；URL资源；File文件资源。
